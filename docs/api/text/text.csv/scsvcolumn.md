A csv column.


## Fields

### `Field length:Size_T`

The length of the column.

<br/>

### `Field quoted:Byte`

A non-zero value indicates that the column is quoted.

<br/>

## Methods

### `Method GetValue:String()`

Returns the value of the column, or [Null](../../../brl/brl.blitz/#null) if the column is empty.

<br/>
